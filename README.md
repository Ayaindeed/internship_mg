# Plateforme de Gestion des Stages
Application web pour gérer les stages au Maroc. Permet aux étudiants de trouver des stages et aux entreprises de recruter des stagiaires.
Services:
- Gestion des stages
- Gestion des étudiants
- Gestion des entreprises

Tech utilisées:
- Jakarta EE
- API REST
- Base de données avec Hibernate

Principales fonctions:
- Créer et suivre les stages
- Gérer les profils étudiants
- Gérer les entreprises partenaires
- Valider les stages
- Suivre les candidatures

Une application simple et efficace pour connecter étudiants et entreprises dans le cadre des stages.
